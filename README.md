## 🎮 遊び方

### 準備するもの
- パソコンまたはスマートフォン（Chrome / Safari 推奨）
- Webカメラ
- マイク

### 手順

1. **ミッション開始**
   - ブラウザでアクセスし、カメラとマイクを許可します
   - 画面中央のボタン、またはスペースキー（PCのみ）、画面タップでミッションを開始します

2. **怪物をロックオン**
   - 画面には巨大な怪物が街を破壊しているシーンが表示されます
   - 右上のHUDにカメラ映像がリンクされます

3. **絆（KIZUNA）を同期する**
   - 手で「キツネ」の形を作ります（影絵のキツネのポーズ）
   - 画面に現れるプロファイル（横向き）のシルエットに自らの手を重ねてください
   - 同期が完了すると「BOND_LINKED」状態となり、周囲の時間が止まったかのようなスタンバイ状態に入ります

4. **「コン！」と喚ぶ**
   - 準備ができたら、合言葉「コン！」を唱えてください
   - 声が認識されるか、画面をタップすることでキツネの悪魔を召喚します
   - キツネの悪魔が巨大な牙で怪物を一撃で仕留めます！

5. **勝利と平和**
   - 怪物は討伐され、歓喜に包まれる街（dead_bug.jpg）が表示されます
   - 英雄としての任務を果たしたキツネが静かに佇み、煙と共に消えていきます

6. **次なる任務へ**
   - 「TAP TO RESTART」またはスペースキーでいつでもミッションをリプレイ可能です

## ✨ V2 の新機能

- **HERO アライアンス・テーマ**: 全体のUIを攻撃的な赤から、ヒーローらしい「シアン＆ブルー」のテックデザインに統一。
- **直感的なハンドガイド**: SVG図形から、実際の手の形に基づいた透過シルエット画像に変更し、初心者でも迷わずポーズを作れるように改善。
- **モバイル最適化**:
    - 全画面タップによるミッション進行（スタート・召喚・リトライ）。
    - 音声認識エンジンの強化：スマホ特有の誤変換（「今」「婚」「come」など）や音パターンを考慮した高度なマッチング。
- **シネマティックHUD**: 没入感を高めるため、操作ガイドを最小限のハイテクなHUDデザインにブラッシュアップ。

## 📸 シーン紹介

- **DECAYED_CITY**: 怪物が暴れる戦場。
- **BOND_SYNC**: ユーザーの手とキツネの意識をリンクさせる神聖な儀式。
- **SUMMON_EX**: 画面全体を覆うキツネの牙が怪物を喰らい尽くす！
- **PEACE_RETURNED**: 勝利後、怪物の死骸を背景に平和が戻った街。
