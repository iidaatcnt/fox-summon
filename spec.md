# FOX SUMMON (狐召喚) 仕様書

## 1. プロジェクト概要
「チェンソーマン」に登場する早川アキの「狐の悪魔」召喚アクションを再現したジョークWebアプリ。
カメラに向かって「狐の影絵（ハンドサイン）」を作り、「コン！」と唱えることで、AR演出と共に狐の悪魔を召喚する。

## 2. コンセプト & ターゲット
*   **コンセプト**: "厨二病"心をくすぐる、言霊とジェスチャーによる没入型AR体験。
*   **利用シーン**: 居酒屋、パーティー、友人と集まっている時の「一発芸」。
*   **プラットフォーム**: Webブラウザ (PC / Mobile)。インストール不要でURLアクセスのみで動作。

## 3. 機能要件

### 3.1. ハンドサイン検知 (The Fox Hand)
*   **技術**: MediaPipe Hands
*   **トリガー**: 以下の形状を「狐サイン」として認識する。
    *   親指、中指、薬指の先端が近接している（狐の口）。
    *   人差し指、小指が上方向に伸びている（狐の耳）。
*   **アクション**: 検知中、画面上に「LOCK ON」インジケーターを表示し、追従させる。

### 3.2. 音声認識 (Voice Trigger)
*   **技術**: Web Speech API
*   **トリガーワード**: 「コン」「こん」「KON」
*   **アクション**: 「LOCK ON」状態でトリガーワードを検知した場合のみ、召喚シーケンスへ移行する。
*   **備考**: 周囲の雑音対策として、ハンドサイン検知中のみ音声認識結果を有効にする（誤爆防止）。

### 3.3. 召喚演出 (Summoning Effect)
*   **技術**: Three.js (@react-three/fiber, @react-three/drei)
*   **演出**:
    *   巨大な狐の頭部モデル（またはビルボード）が画面奥から出現。
    *   画面全体が激しく振動する（Camera Shake）。
    *   衝撃と共に「KON!」の文字エフェクト。

### 3.4. 対話モード (Dialogue)
*   召喚後、狐の悪魔からのメッセージを表示。
    *   テキスト: 「...これは昼の悪魔だね。飲み込んでいい？」
    *   ユーザーアクション: 再度「コン」と言うか、画面タップで肯定（リセットまたは捕食演出へ）。

## 5. UI/UX フロー

1.  **IDLE (待機)**: カメラ起動。通常のパススルー映像。「SYSTEM ONLINE」等のHUD表示。
2.  **SCANNING (探索)**: 手を検出したが、狐サインではない状態。
3.  **LOCKED (ロックオン)**: 狐サイン検出。ターゲットサイトが手に吸い付く。マイクアイコンが活性化。
4.  **SUMMON (召喚)**: 音声「コン」を検知。3Dエフェクト発動。
5.  **DIALOGUE (対話)**: テキスト表示。次の入力を待つ。
